# AI-Powered Rural Supply Chain & Price Intelligence System

## Overview
This project aims to improve farmer profitability by predicting mandi prices, forecasting crop demand trends, and recommending optimal selling markets using AI-based analytics.

## Key Features
- Mandi price prediction using regression models
- Demand trend forecasting
- Optimal market recommendation
- Dashboard-based visualization

## Technology Stack
- Python
- Machine Learning (Regression & Time Series)
- REST API (Flask/FastAPI)
- Database (PostgreSQL/SQLite)

## Impact
- Reduces price uncertainty
- Increases farmer income potential
- Strengthens rural supply chain efficiency
